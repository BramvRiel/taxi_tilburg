# taxi_tilburg

Ik denk aan taxi's laten reiden naar centraal gelegen locatie, waarschijnlijk station. Ten gunste van taxibedrijf.
Of, kijken naar reeds geplande routes. Alvast klaar gaan staan. Ten gunste van reiziger.

- locaties
- locaties_afstanden
- locaties_reistijden
- reizigers

### UI:
#### Excel import pagina
+ formulier
+ resultaat
#### route maker pagina
+ vertrektijd
+ route bouwer
+ aankomsttijd
+ totaal km
+ totaal h m s
